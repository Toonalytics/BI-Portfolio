Overview

This project focuses on revealing insights about the different content types on the movie streaming service – Netflix.

Findings

To kick things off, I explored a Netflix dataset from Kaggle, containing about 9,000 rows of data with details such as Titles, Directors, Cast, Release Year, Genre, TV Ratings, Category (TV Show/Movie), Duration, and Descriptions.

My goal was to answer key questions like:

  • How is the content distributed by category?
  • What’s the release trend over the years?
  • Which genres, directors, and cast members dominate by release count?
  • How many shows fall into each TV rating?
  • What are the most common keywords across titles?... amongst others.

A country-specific breakdown of all these insights would’ve been time-intensive, but slicers came to the rescue for easy filtering and dynamic exploration!

The main challenge was handling null values and messy data, but the biggest surprise? Indian actors dominated the Cast segment, even more than their Western counterparts.
